# Rash Cigarettes UI

A mini e-commerce UI project called "Rash Cigarettes", built with React + Vite.
This project focuses on modern UI layout, responsive design, and reusable component structure for frontend practice.

## 🚀 Live Demo
🔗 <a href="https://rash-cigarettes.vercel.app" target="_blank">https://rash-cigarettes.vercel.app</a>

## 🛠 Tech Stack
- React 18
- Vite
- React Router DOM
- React Icons
- React Toastify
- CSS (Custom Styling)

---

## 📥 Clone This Repository

```bash
git clone https://github.com/mfarrass/rash-cigarettes.git
cd rash-cigarettes
```

---

## 📦 Installation

```bash
npm install
```

---

## ▶️ Run Development Server

```bash
npm run dev
```

App will run on:

```
http://localhost:5173
```

---

## 🏗 Build for Production

```bash
npm run build
```
