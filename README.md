# Rash Cigarettes UI

Modern e-commerce "Rash Cigarettes" UI built using React + Vite.

## 🚀 Live Demo
🔗 https://rash-cigarettes.vercel.app

## 🛠 Tech Stack
- React 18
- Vite
- React Router DOM
- React Icons
- React Toastify
- CSS (Custom Styling)

---

## 📥 Clone This Repository

```bash
git clone https://github.com/mfarrass/rash-cigarettes.git
cd rash-cigarettes
```

---

## 📦 Installation

```bash
npm install
```

---

## ▶️ Run Development Server

```bash
npm run dev
```

App will run on:

```
http://localhost:5173
```

---

## 🏗 Build for Production

```bash
npm run build
```
